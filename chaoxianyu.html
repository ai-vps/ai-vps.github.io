전 세계적으로 가장 널리 사용되는 클라우드 인프라 플랫폼입니다. 엔터프라이즈급 SLA, 글로벌 리전, 포괄적인 AI/ML 서비스를 제공합니다. 대규모 워크로드와 프로덕션 환경에 적합합니다.

Amazon Web Services

  • Bedrock: 서버리스 Foundation Model API

하이퍼스케일러 선택 기준

하이퍼스케일러는 대규모 워크로드, 엔터프라이즈 SLA, 글로벌 배포가 필요할 때 최적입니다. 다만 GPU 가격이 네오클라우드 대비 60~150% 높을 수 있으며, VPC 설정, IAM 구성, 할당량 승인 등 사전 작업이 필요합니다. 소규모 프로젝트나 스타트업이라면 아래 뉴 클라우드나 GPU 클라우드 섹션을 먼저 확인하세요. (citation:5)

국내 클라우드가 필요한 경우

데이터 주권, 한국어 지원, 원화 결제, 세금계산서 발행이 중요한 국내 기업이나 대학·연구기관이라면 국내 GPU 클라우드를 먼저 살펴보세요. 의료·금융·공공 분야는 해외 데이터 반출이 어려울 수 있어 국내 리전이 필수적입니다. 가격은 해외 네오클라우드보다 다소 높을 수 있으나, 기술 지원과 규정 준수 측면에서 장점이 있습니다. (citation:5)

PAGE 09 — DEPLOYMENT GUIDE

AI 인프라 배포 가이드

초보자도 따라할 수 있는 단계별 배포 가이드입니다. VPS 선택부터 AI 에이전트 실행까지, 핵심 단계를 안내합니다.

1단계: 용도에 맞는 인프라 선택

가장 중요

프로젝트의 규모와 목적에 따라 적합한 인프라를 먼저 선택하세요. 잘못된 선택은 비용과 시간을 낭비할 수 있습니다.

  • AI 모델 학습/파인튜닝: GPU 클라우드 (VESSL, RunPod, Lambda)
  • LLM API 활용: AI 서비스 (OpenAI, Anthropic, Google AI)
  • AI 에이전트 호스팅: VPS (Hetzner, Oracle Free) 또는 SaaS (Kimi Claw)
  • 프로덕션 서비스: 하이퍼스케일러 (AWS, GCP, Azure)
  • 예산 제한: Oracle Cloud Free, Vultr, Railway
  • 국내 규정 준수: 국내 클라우드 (NCP, KT Cloud, Naver Cloud)

2단계: 서버 생성 및 접속

5~10분

선택한 플랫폼에서 서버를 생성하고 SSH로 접속합니다. 대부분의 네오클라우드는 가입 후 5~10분 안에 서버를 사용할 수 있습니다. (citation:5)

  • OS 선택: Ubuntu 22.04 LTS 권장
  • 사양 확인: 최소 2vCPU/4GB RAM (OpenClaw 기준)
  • SSH 키 등록 및 방화벽 설정
  • 한국 리전 또는 가까운 리전 선택 (지연 시간 최소화)
  • 정기 백업 설정 권장

3단계: 시스템 업데이트 및 AI 프레임워크 설치

10~20분

서버 접속 후 시스템을 업데이트하고 필요한 AI 프레임워크를 설치합니다. (citation:4)

  • 시스템 업데이트: apt update && apt upgrade -y
  • Docker 설치: curl -fsSL https://get.docker.com | sh
  • OpenClaw 설치: 공식 설치 스크립트 실행

AI 인프라를 실제로 어떻게 활용하는지, 그리고 가장 많이 묻는 질문에 대한 답변을 정리했습니다.

LLM API로 AI 앱 빌딩

OpenAI GPT-4o, Anthropic Claude 4, Google Gemini API를 활용하여 챗봇, 문서 요약, 코드 생성 등 다양한 AI 애플리케이션을 빌드합니다. API 키만 있으면 바로 시작할 수 있습니다.

  • 추천 인프라: AI 서비스 API + VPS (백엔드)
  • 예산: API 토큰 과금 + VPS 월 ~1만원
  • 난이도: 초급

자체 AI 모델 학습

자체 데이터로 AI 모델을 파인튜닝하거나 처음부터 학습합니다. H100, A100 GPU 클러스터가 필요하며, 대규모 학습에는 멀티노드 인프라가 필수입니다.

  • 추천 인프라: GPU 클라우드 (VESSL, RunPod, CoreWeave)
  • 예산: $2~7/hr/GPU (학습 시간에 따라)
  • 난이도: 중~상급

AI 에이전트 자동화

OpenClaw, Claude Code 등 AI 에이전트를 활용하여 자동화 워크플로를 구축합니다. 이메일 분석, 리서치, 코드 리뷰, 보고서 작성 등을 자동화할 수 있습니다. (citation:3)

  • 추천 인프라: SaaS (Kimi Claw) 또는 VPS 셀프 호스팅
  • 예산: 무료~월 수만원
  • 난이도: 초~중급

AI 보안 테스팅

AI 기반 보안 펜테스팅 도구를 활용하여 인프라의 취약점을 검증합니다. 자율 노출 검증, 에이전트 공격 워크플로, CI/CD 통합 테스트 등 다양한 카테고리가 있습니다. (citation:2)

  • 추천 도구: Horizon3.ai, Penligent, XBOW
  • 예산: 도구별 상이 (무료~엔터프라이즈)
  • 난이도: 중급

자주 묻는 질문 (FAQ)

AI 개발에 최소 얼마가 필요한가요?

무료로 시작할 수 있습니다. Oracle Cloud Always Free로 ARM 서버를 무료로 쓸 수 있고, OpenAI와 Google AI는 무료 티어를 제공합니다. 월 5만원 이하로도 충분히 AI 에이전트를 운영할 수 있습니다. (citation:4)

GPU 클라우드와 VPS의 차이는?

GPU 클라우드는 AI 모델 학습·추론에 특화된 고성능 GPU(A100, H100)를 제공합니다. VPS는 범용 서버로, AI 에이전트 호스팅, 웹 앱, API 서버 등에 사용됩니다. GPU가 필요 없는 AI API 활용에는 VPS로 충분합니다. (citation:5)

국내 클라우드가 해외보다 비싼가요?

GPU 가격 기준으로는 해외 네오클라우드(VESSL Cloud 포함)가 하이퍼스케일러 대비 60~80% 저렴합니다. 국내 클라우드는 가격이 다소 높을 수 있으나, 한국어 기술 지원, 원화 결제, 데이터 주권 측면에서 장점이 있습니다. (citation:5)

AI 에이전트 호스팅은 어떻게 하나요?

두 가지 방법이 있습니다. 1) SaaS 플랫폼(Kimi Claw): 원클릭 배포, 설정 불필요. 2) 셀프 호스팅(VPS): Hetzner나 Oracle Cloud에서 서버를 만들고 OpenClaw를 직접 설치합니다. 초보자는 SaaS를, 제어권이 필요하면 셀프 호스팅을 추천합니다. (citation:3)(citation:10)

OpenClaw를 구동하려면 RAM이 얼마나 필요한가요?

최소 4GB RAM을 권장합니다. 2vCPU/4GB RAM이면 기본적인 AI 에이전트 워크플로를 실행할 수 있으며, 복잡한 멀티 에이전트 작업에는 8GB 이상을 추천합니다. (citation:4)

하이브리드 전략이란 무엇인가요?

기존 클라우드(AWS/GCP)에서 데이터 파이프라인을 운영하면서, GPU 학습 작업만 네오클라우드(VESSL, RunPod)로 분리하는 전략입니다. 비용 효율성과 유연성을 동시에 확보할 수 있습니다. (citation:5)
